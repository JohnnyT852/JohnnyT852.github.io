Game

For my game project, I took the already existing game and added a few things to it. In my game when you get to level two, these small squares will start flashing all over the screen causing somewhat of a distraction. My original idea was to have this as sort of an unfair game to where if you touch the squares, you lose the game. However that type of coding is probably way above my level as it was very challenging to put the flashing squares there in the first place. So these squares simply serve as just a distraction then.
